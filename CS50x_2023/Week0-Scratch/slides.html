未来会发生什么

  • 本周您将学习 Scratch，一种可视化编程语言。在接下来的几周里，您将学习 C。看起来像这样： 
      #include "stdio.h"

  int main(void)
  {
      printf("hello, world\n");
  }
  • 了解算法，学习有关记忆的知识，了解有缺陷的代码以及导致计算机崩溃的原因，了解哈希表等数据结构。
  • 然后，过渡到一种新的、更高级的语言，称为Python。您的代码将如下所示： 
      print("hello, world")
  • 本课程将使您深入了解最新的编程语言是如何从早期的语言发展而来的。
  • 我们还将研究如何使用数据库和第三方框架来构建 Web 应用程序。
